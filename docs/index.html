ServerChannelChatへようこそ

このプラグインは、複数のMinecraftサーバー間でチャットを同期するためのものです。BungeeCord、Velocity、Paper、Spongeなど、さまざまなサーバープラットフォームに対応しています。

主な特徴

  • クロスプラットフォーム対応: 主要なプロキシおよびサーバーソフトウェアで動作します。
  • 柔軟なチャンネル設定: サーバーごとや権限ごとにチャットチャンネルを自由に設定できます。
  • 多機能: プライベートメッセージ、フォーマットのカスタマイズ、ミュート機能などを搭載しています。

クイックスタート

プラグインを始めるには、まずサーバーにインストールする必要があります。インストールが完了したら、以下の基本的なコマンドを試してみましょう。

/ch create §b<ChannelName>

このコマンドは、チャンネルを作成します。詳細についてはコマンドタブを確認してください。

warning

注意

このプラグインは現在ベータ版です。本番環境で使用する前に、必ずデータのバックアップを取ってください。
