/ch create

このコマンドはチャットチャンネルに参加します。

使用例

/ch join §b<ChannelName>

説明

  • <ChannelName>: 参加するチャットチャンネルの名前を指定します。
  • チャンネル名には特殊文字を使用できません。
warning

注意

パスワードが設定してある場合、入力が必要になります。

/ch g arrow_forward
